## Visualizar
 - No seu terminal, clone o respositório:
   - ``` git clone git@github.com:marcusrodriguesdev/todo-list.git ```
 - Entre no diretório do projeto
   - ``` cd todo-list```
 - Na pasta raíz, instale as dependências:
   - ``` npm install ```
 - E inicialize
   - ``` npm start ```
